# Доровской Алексей Васильевич

Написать map, защищенную мьютексом и тест для нее.
От кода не требуется быть production ready. 

В тесте у мапы ключи числовые.
Количество ключей должно быть равно году рождения какой-нибудь известной личности или году известного события.
В комментарии к числу указать, откуда взят год.
Соответственно диапазон ключей от единицы до года.
Заранее мапу ключами и значениями заполнять нельзя.
В тесте надо запустить четыре гороутины.

Каждая гороутина делает следующее:

1) обращается по ключу к значению мапы. Если значения нет, то оно должно быть создано.
2) увеличивает значение на 1
Важное замечание: мапа не должна иметь функции инкремента значения.

В конце теста после завершения работы всех гороутин (внимание!) по каждому ключу должно лежать число 3 (три!).
Гороутины не должны обращаться к ключам упорядочено в порядке возрастания или убывания.

В мапу еще надо добавить счетчик обращения к ключам и счетчик добавления ключей.
Счетчик обращения в итоге должен быть равен год * 3, счетчик добавления - году.

В комментарии к коду напишите ФИО, чтобы не возникло путаницы.
